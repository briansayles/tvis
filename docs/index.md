# TourneyVision

## About

TourneyVision is a poker tournament management app that you can use to organize and run
your own home poker tournaments. It helps you to:

* Create a blinds schedule
* Keep track of the time (everyone can see what the current and next blinds are, and for how long)
* Customize the messages to be spoken at 1 minute before the end of each round and at the end of each round
* Create a chips schedule
* Keep track of what chips are (or should be) in play vs. which ones can/should be colored up
* Create a buyin schedule (eg: $20 for 1000 chips; $20 rebuy for an additional 1000 chips; $5 add-on for $500 additional chips, etc...)
* Keep track of who bought in, rebought, added on, etc...
* Create a payout schedule
* Keep track of who got paid and how much
* Keep track of statistics for players over multiple tournaments (FUTURE)
* Keep track of tables and player seating (FUTURE)
* Have seats assigned randomly at the start of a tournament or when combining tables (FUTURE)
